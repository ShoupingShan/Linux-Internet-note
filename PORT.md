# 端口(TCP包头)
### TCP协议包头
    源端口16位  (2^16)
    目的端口16位  (2^16)

### UDP协议包头
    源端口16位
    目的端口16位


## 常见端口号作用 (重要)
    FTP(文本传输协议) :      端口号  20(数据传递)   21(传输命令)
    SSH(安全shell协议) :    端口号  22
    telnet(远程登录协议) :   端口号  23(禁止开启,未加密)
    DNS(域名系统) :         端口号  53
    http(超文本传输协议) :   端口    80
    SMTP(简单邮件传输协议) : 端口号  25
    POP3(邮局协议3代) :     端口号   110

* 查看本机启用的端口

      netstat -an
      选项:
         -a:  查看所有连接和监听端口
         -n:  显示IP地址和端口号,而不显示域名和服务器
